# JS Browser Music

I just casually found out you can write javascript code that when entered into the chromium browser console plays notes, so I had some fun around it and recreated some songs with javascript code.

# How to Run (Only availible on desktop browsers rn)
1. Press F12 in your browser
2. Select console on the top bar, next to "Sources" and "Elements"
3. Type "allow pasting" into the console
4. Paste the code
5. Here your browser sing :)
